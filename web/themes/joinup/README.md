# Frontend tools

## Requirements

* Node.js with its `npm` command

## Local development setup

* open a shell and go to joinup theme folder:
```Shell
$ cd web/themes/joinup
```
* install local node dependencies:
```Shell
$ npm install
```
* to compile scss files:
```Shell
$ npm run css
```
* to watch scss files:
```Shell
$ npm run watch
```
